Package org.ajoberstar.grgit.operation

User facing operations classes. Should only be used when obtained from a Grgit instance.

  • Class Summary 
    Class Description
    AddOp Adds files to the index.
    ApplyOp Apply a patch to the index.
    BranchAddOp Adds a branch to the repository.
    BranchChangeOp Changes a branch's start point and/or upstream branch.
    BranchListOp Lists branches in the repository.
    BranchRemoveOp Removes one or more branches from the repository.
    BranchStatusOp Gets the tracking status of a branch.
    CheckoutOp Checks out a branch to the working tree.
    CleanOp Remove untracked files from the working tree.
    CloneOp Clones an existing repository.
    CommitOp Commits staged changes to the repository.
    FetchOp Fetch changes from remotes.
    InitOp Initializes a new repository.
    LogOp Gets a log of commits in the repository.
    MergeOp Merges changes from a single head.
    OpenOp Opens an existing repository.
    PullOp Pulls changes from the remote on the current branch.
    PushOp Push changes to a remote repository.
    RemoteAddOp Adds a remote to the repository.
    RemoteListOp Lists remotes in the repository.
    ResetOp Reset changes in the repository.
    RevertOp Revert one or more commits.
    RmOp Remove files from the index and (optionally) delete them from the working tree.
    StatusOp Gets the current status of the repository.
    TagAddOp Adds a tag to the repository.
    TagListOp Lists tags in the repository.
    TagRemoveOp Removes one or more tags from the repository.
